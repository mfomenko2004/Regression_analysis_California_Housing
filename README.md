## 📌 Описание проекта

Данный проект представляет собой комплексный анализ датасета **California Housing** с целью прогнозирования медианной стоимости жилья. В работе реализованы и сравнены несколько методов регрессионного анализа, проведена диагностика мультиколлинеарности и остатков моделей.

### Датасет
- **Источник:** `sklearn.datasets.fetch_california_housing`
- **Наблюдений:** 20 640
- **Признаков:** 9
- **Целевая переменная:** MedHouseVal — медианная стоимость жилья (в единицах $100 000)

---

## 🔧 Реализованные методы

| Метод | Описание |
|-------|----------|
| **OLS** | Обычный метод наименьших квадратов (Scikit-learn + Statsmodels) |
| **Ridge** | L2-регуляризация |
| **Lasso** | L1-регуляризация (автоматический отбор признаков) |
| **ElasticNet** | Комбинированная регуляризация (L1 + L2) |
| **Кросс-валидация** | 10-кратная, с KFold |
| **GridSearchCV** | Подбор гиперпараметров (α, l1_ratio) |
| **VIF-анализ** | Диагностика мультиколлинеарности |
| **Диагностика остатков** | Q-Q plot, гистограмма, тест Шапиро-Уилка |

## 📈 Результаты

### Сравнение моделей

| Модель | R² | RMSE | MAE |
|--------|----|------|-----|
| OLS | 0.5758 | 0.7456 | 0.5332 |
| Ridge | 0.5758 | 0.7455 | 0.5332 |
| Lasso | 0.5769 | 0.7446 | 0.5331 |
| ElasticNet | 0.5767 | 0.7448 | 0.5331 |

### VIF-анализ (мультиколлинеарность)

| Признак | VIF | Статус |
|---------|-----|--------|
| Longitude | 641.22 | ✘ высокая |
| Latitude | 560.58 | ✘ высокая |
| AveBedrms | 48.33 | ✘ высокая |
| AveRooms | 46.79 | ✘ высокая |
| MedInc | 11.83 | ✘ высокая |
| HouseAge | 7.16 | ⚠ умеренная |
| Population | 2.92 | ✔ норма |
| AveOccup | 1.08 | ✔ норма |

### Ключевые выводы

1. **OLS** показал базовое качество (R² ≈ 0.576), но VIF-анализ выявил высокую мультиколлинеарность географических признаков (Latitude/Longitude) и признаков, связанных с комнатами (AveRooms/AveBedrms).

2. **Регуляризация** (Ridge, Lasso, ElasticNet) не привела к значительному улучшению R², но обеспечила более стабильные коэффициенты.

3. **Диагностика остатков** выявила отклонение от нормальности (тест Шапиро-Уилка: p < 0.05), что типично для больших выборок — центральная предельная теорема обеспечивает асимптотику.

4. **Рекомендуемая модель:** Lasso — лучший баланс между предсказательной способностью (R² = 0.5769) и интерпретируемостью.
